# xray-pilot v0.4.0

节点接入流程产品化，订阅信息页一批可读性优化，并为未来的实时控制面（v0.4.5）与可能的 agent 化（v0.7.0+）预留扩展点。

## 亮点

### 节点接入产品化（一键脚本接入）

把"手工 SSH 装机 + 回面板新增节点"两段流程合并为一行 curl 命令：

1. **面板新增「一键接入」对话框**：填节点名 / 地区 / 所有者 / SSH 参数 / TTL，后端生成一次性安装 token，返回完整 curl 命令。
2. **`scripts/node-bootstrap.sh` 自动模式**：检测 `PANEL_URL` + `INSTALL_TOKEN` 环境变量后自动从面板拉公钥写入 `authorized_keys`、装 xray、配 BBR、重启 SSH，最后 POST 回填节点元数据。原有交互模式向后兼容、README 旧用法不影响。
3. **安全模型**：token 32 字节随机 + 默认 10 分钟 TTL + 一次性（used 后 410 Gone）+ 首次访问 IP 绑定（异地访问 403）+ 调度器每小时清理过期未使用 token。

完整流程从原 5-7 步手工降级为：管理员复制命令 → 目标机器执行 → 等节点出现在列表。**panel 必须配置 `ssh.default_key_path` 且对应 `.pub` 公钥可读**，前端在打开对话框前由后端预校验。

### 为未来演进打的地基

- `Node.connection_mode` 字段（v0.4.0 全填 `ssh`），为 v0.7.0+ 的可能 agent 化预留切换位
- `Node.registered_at` 时间戳，区分"一键接入"与"手工新增"两种接入路径
- `/api/install/register` 接口 payload 设计为接入模式无关，将来 agent 启动可复用同一端点
- 操作日志 actor 字符串约定（`admin:<id>` / `system:install-token:<前8位>` / 未来 `system:agent:<node_id>`），v0.5.0 多管理员落地时可平滑迁移

## 实现要点

- **新表 `node_install_tokens`**：GORM AutoMigrate 自动建表，含 token / node_meta(JSON) / expires_at / used_at / used_by_ip / node_id 字段
- **新路由组 `/api/install/*`**：与 JWT 域并列，使用 install token query 参数鉴权；脚本不需要也不应持有 admin 凭据
- **公钥推导**：从 `config.ssh.default_key_path` + `.pub` 自动定位，不增加新配置项
- **调度器接入**：固定 1 小时清理一次过期未使用 token，启动后 5 分钟首次触发
- **单测覆盖**：`TestCreateToken_RequiresPanelPubkey` / `TestCreateToken_RejectsDuplicateName` / `TestAuthorizeToken_OneShotAfterRegister` / `TestAuthorizeToken_IPBinding` / `TestAuthorizeToken_Expired` / `TestCleanupExpired_RemovesUnusedExpiredOnly` / `TestBuildInstallCurlCommand_FormatsPanelAndToken`，覆盖所有关键安全不变量

## 兼容性

- **数据库**：新增 `node_install_tokens` 表 + `nodes` 表加 `connection_mode` / `registered_at` 两列，AutoMigrate 自动处理，老数据零侵入
- **现有节点接入流程不变**：手工新增 + 既有交互式 bootstrap 模式继续可用
- **API**：新增端点，不破坏既有端点契约

## 升级建议

直接覆盖升级即可。升级后可在 panel 上点「一键接入」体验自动化接入流程。

## 风险与权衡（值得知道）

- **curl 命令包含 token**：截图泄露后理论可被异地使用 — 已通过 IP 绑定 + 短 TTL + 一次性缓解。建议管理员避免在共享屏幕上展示 token。
- **panel-pubkey 端点匿名时被探测**：通过 token query 鉴权 + 32 字节熵抵御。
- **bootstrap 注册回 panel 失败**：脚本仅 WARN 不退出，xray 已正常起；管理员可手工新增节点录入 IP 完成接入。
