# xray-pilot v0.4.0

节点接入产品化，订阅信息页可读性优化。

## 新增

### 节点一键接入

面板节点列表新增「一键接入」按钮，填写节点元数据后生成一行 curl 命令。在新节点上以 root 执行后，脚本自动完成 SSH 配置、xray 安装、BBR 调优并回填节点信息到面板，**不再需要手填公钥、也不用回面板新增节点**。

Token 一次性、绑定首次访问 IP、默认 10 分钟过期，可在对话框中选 1 小时 / 24 小时。`scripts/node-bootstrap.sh` 的原有交互模式保持不变，老用法不受影响。

### 订阅信息页优化

- **节点协议徽章**：节点行展示 VLESS+Reality / VLESS+WS+TLS / Trojan / Hysteria2 四类配色徽章，与后台一致
- **最后同步时间**：显示「同步于 5 分钟前」等相对时间，超过 24 小时未同步会标黄提示
- **节点备注**：`Node.Remark` 非空时透出，方便给出使用说明
- **Clash 一键导入**：备用订阅链接的 Clash 系新增 `clash://install-config` 按钮，移动端 Mihomo / Clash Verge / ClashX 直接拉起 App

## 修复

- **复制按钮在 HTTP 部署下可用**：之前在 `http://192.168.x.x` 等内网地址访问面板时，所有复制按钮（订阅链接、节点配置预览、QR 弹窗、一键接入命令）会静默失败；现在自动回退到兼容方式，HTTP 部署也能正常复制
- **系统设置改 SSH 私钥路径立即生效**：「默认 SSH 私钥」改路径后，一键接入立刻识别新路径，不再需要改 `config.yaml` 重启
- **诊断同时校验公钥**：「默认 SSH 私钥」诊断项现在会校验 `.pub` 公钥是否存在，缺失时给出 `ssh-keygen` 修复命令，避免「诊断显示正常但一键接入仍报错」
- **控制台不再被 GORM 噪音刷屏**：业务上的「未找到记录」（如 setting 走默认值）不再以 warning 形式输出

## 升级

- 数据库 schema 自动迁移（GORM AutoMigrate），无需执行脚本，老数据无侵入
- 无破坏性 API 变更，可直接覆盖二进制重启

## 注意事项

- 一键接入命令包含临时 token，避免在共享屏幕展示或截图分享。Token 已设短 TTL + IP 绑定，但仍属敏感凭据
- 使用一键接入前请确认面板已配置 `ssh.default_key_path` 且对应 `.pub` 公钥可读，否则生成 token 会被拒绝
