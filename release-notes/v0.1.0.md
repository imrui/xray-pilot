# xray-pilot v0.1.0

首个公开版本。面向小团队的自托管 Xray 管理面板，前后端打包为单个 Go 二进制，可直接在 Linux 上部署。

## 新增

- **节点管理**：新增 / 编辑 / 启用禁用 / 删除，支持 SSH 连通性测试、单节点与批量同步、配置预览
- **协议与密钥**：协议配置 CRUD，节点级协议密钥生成与维护
- **用户与分组**：用户与分组 CRUD，订阅链接重置
- **运维能力**：订阅链接生成、配置漂移检测、节点健康检查、操作日志
- **系统配置**：部分系统设置可在面板内修改并持久化

## 发布产物

Linux amd64 / arm64、macOS amd64 / arm64 预编译二进制，默认服务端口 `2026`，提供 Linux 一键安装脚本。
