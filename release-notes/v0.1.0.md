## xray-pilot v0.1.0

`xray-pilot` 首个公开版本发布。

这是一个面向小团队的自托管 Xray 管理面板，支持节点管理、协议配置、用户与分组管理、订阅分发，以及基础的运维巡检能力。项目采用前后端一体化设计，前端资源已内嵌到 Go 二进制中，便于在 Linux 服务器上直接部署。

### 本版本功能

- 节点管理
  - 支持节点新增、编辑、启用/禁用、删除
  - 支持 SSH 连通性测试
  - 支持单节点和批量同步配置
  - 支持配置预览
  - 支持节点地址变化后自动清理旧 `known_hosts` 记录

- 协议与密钥管理
  - 支持协议配置的新增、编辑、删除
  - 支持节点级协议密钥生成与维护
  - 支持协议与节点联动下发

- 用户与分组管理
  - 支持用户新增、编辑、启用/禁用、删除
  - 支持分组管理与节点关联
  - 支持订阅链接重置等常用操作

- 订阅与运维能力
  - 支持用户订阅链接生成
  - 支持配置漂移检测
  - 支持节点健康检查
  - 支持操作日志查看
  - 支持部分系统运行时配置写入数据库并通过 Web UI 管理

### 前端与交互

- 完成后台管理界面的首版 UI/UX 重构
- 优化用户、节点、分组、协议等页面的操作流
- 引入抽屉式编辑、操作菜单、搜索筛选等交互方式
- 统一确认弹窗、表单结构和控制台视觉风格

### 部署与发布

- 提供单二进制运行模式，前端资源已内嵌
- 默认服务端口为 `2026`
- 提供 Linux 一键安装脚本
- 新增 GitHub Actions 自动构建与 Release 发布流程

### 发布产物

本次 Release 提供以下平台的预编译二进制：

- Linux amd64
- Linux arm64
- macOS amd64
- macOS arm64

### 说明

`v0.1.0` 是项目的第一个正式版本，适合用于自建 Xray 管理和轻量运维场景。后续版本会继续完善交互体验、配置能力、安装流程和运维工具链。
