# xray-pilot v0.2.2

新增 SQLite 数据库定时备份与备份文件管理界面。

## 新增

- 每 24 小时（可配）自动生成 `xray-pilot-YYYYMMDD-HHMMSS.db` 完整快照（`VACUUM INTO`），对运行服务不产生写阻塞
- 按保留天数（默认 30 天）自动清理过期备份
- 系统设置页新增「数据备份」面板：备份列表（名称 / 大小 / 创建时间）、立即备份、单文件下载与删除
- 备份目录、调度间隔、保留天数全部可在 UI 修改并即时生效

## 注意事项

- **仅支持 SQLite**：Postgres 用户请继续使用 `pg_dump`
- **本地存储**：备份与服务进程同主机，**强烈建议自行通过 cron / rsync / 对象存储客户端**把 `data/backup/` 定期拉到异地
- **不含自动恢复**：恢复需停服 → 替换 `xray-pilot.db` → 重启
- **MasterKey 需另外保管**：备份只是 db 文件本身，Reality 私钥仍以 AES-GCM 加密形式存储；丢失 MasterKey 后即便有 db 备份也无法解出私钥
