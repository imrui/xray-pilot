# xray-pilot v0.3.1

三处前端修复。

## 修复

- **暗色主题下下拉选项不再白底白字**：节点筛选、分页、表单 SelectField 等所有原生 `<select>` 弹层在暗色模式下显示正常
- **节点密钥抽屉的列表高度自适应**：左侧节点列表撑满抽屉可视区域，不再出现「内容未到底已显示滚动条」的间隙
- **节点行 IP 过长不再遮盖端口 / 锁状态**：IP 自动截断，端口编号与 🔒 锁图标始终可见
- **备份文件下载恢复鉴权**：备份文件能正常下载，不再因鉴权头缺失被拒导致存成 `download.json`

## 升级

纯前端修复，无 schema 变更，可直接覆盖。
