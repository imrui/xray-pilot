# xray-pilot v0.2.0

订阅模块为 sing-box 客户端提供原生格式支持。

## 新增

- `?format=singbox` 端点现在返回完整可用的 sing-box JSON Profile（含 `log` / `dns` / `inbounds` / `outbounds` / `route` / `experimental.clash_api`），Hiddify / Karing / NekoBox / sing-box GUI 等客户端可直接导入
- 节点 outbound 自动包装为 `selector`（手动切换）+ `urltest`（自动择优，3 分钟测延迟）组合，对应 GUI 客户端的「代理组」
- 覆盖 4 种协议：VLESS+REALITY+Vision、VLESS+WS+TLS、Trojan、Hysteria2
- 响应头补全 `Content-Disposition: attachment; filename="xray-pilot.json"`；Clash 订阅文件名也补全为 `xray-pilot.yaml`

## 升级

- 无需配置变更，订阅链接形态不变
- V2Ray Base64（`?format=v2ray`）与 Clash YAML（`?format=clash`）输出保持原状
- 基线 sing-box 1.8.0；不引入 rule_set 远程依赖，避免首次启动需联网下载 geosite / geoip
