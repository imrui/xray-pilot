# xray-pilot v0.1.3

本版本聚焦于控制台 UI 重构与交互一致性优化，核心功能保持不变。

## 亮点

- 全新控制台风格与布局重构，统一页面视觉语言（侧栏、顶部、表格、抽屉、弹窗）。
- 新增仪表盘页面，强化节点、用户、日志等核心信息的总览能力。
- 交互统一：列表页支持批量操作、状态切换、搜索筛选、分页控制，减少重复点击路径。
- 主题体验优化：亮色/暗色切换更直接，暗色对比和亮色可读性进一步打磨。

## 重点改进

### 页面与导航

- 左侧菜单支持折叠，结构与命名更统一。
- 顶部区域显示“页面标题 + 副标题”，主区减少冗余标题信息。
- 桌面端滚动行为优化：框架固定，仅主内容区滚动，长页面浏览更稳定。

### 表格与状态交互

- 用户、节点、分组、协议、日志页面的列表样式与操作方式统一。
- 状态显示（Badge）重新设计，修复亮色模式下可读性问题，并降低暗色模式刺眼感。
- 分页能力统一：
  - 支持 `10/20/50/100`
  - 日志页默认 `20/页`
  - 其他页面默认 `10/页`

### 表单与抽屉

- 抽屉编辑体验优化，支持脏数据保护与更清晰的字段分组。
- 节点 SSH 参数支持“系统默认 + 节点可选覆盖”模式：
  - 节点可留空 `ssh_key_path`
  - 运行时自动回退到系统配置 `ssh.default_key_path`

### 预览与可读性

- 节点“预览生成配置”弹窗支持更大尺寸与双向滚动，解决长 JSON 显示不全问题。
- 系统设置页右侧“保存状态”显示优化，避免窄栏换行错位。

## 兼容性说明

- 本版本不引入破坏性 API 变更。
- 分组更新接口新增可选 `active` 字段，用于前端开关式状态切换。

## 升级建议

1. 升级后建议清理浏览器缓存并刷新页面，确保新样式与主题状态生效。
2. 若通过 systemd 部署，建议在系统设置中确认 SSH 默认密钥路径仍为期望值（例如 `/etc/xray-pilot/ssh/id_ed25519`）。
