# xray-pilot v0.1.3

控制台 UI 重构与交互一致性优化。

## 新增

- **仪表盘**：强化节点、用户、日志等核心信息的总览
- **统一交互**：列表页支持批量操作、状态切换、搜索筛选、分页控制（10/20/50/100/页，日志默认 20）
- **节点 SSH 参数**：节点可留空 `ssh_key_path`，运行时自动回退到系统默认配置
- **配置预览弹窗**：支持更大尺寸与双向滚动，解决长 JSON 显示不全

## 修复

- 亮色模式下状态徽章可读性、暗色模式下刺眼感
- 长页面浏览时主区滚动稳定性

## 升级

无破坏性变更；分组接口新增可选 `active` 字段。升级后建议清浏览器缓存以拿到新样式。
