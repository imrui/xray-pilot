# xray-pilot v0.2.4

订阅信息页若干 UI 细节修复。

## 修复

- **累计流量改为 2×2 网格布局**：上行 / 下行 / 合计 / 最近更新四项各占一格，标签字号与数值字号完全统一，4 个数字在两列上对齐，整体几何反差更强。
- **二维码弹窗标题居中显示**：主标题与副标题均水平居中，并将 `padding-right` 改为左右对称 `padding`，与右上角关闭按钮形成视觉对称。
- **备用订阅二维码弹窗标题不再被挤换行**：之前把客户端示例 `Hint`（如 `V2RayN / NekoRay / Shadowrocket`）作为 `region-tag` 渲染，长 hint 把标题挤成两列竖排。现在 `Hint` 改为弹窗副标题，主标题独占一行。

## 实现要点

- 累计流量从「上下行 flex 行 + 合计/时间 space-between 行」改为单一 `traffic-grid`（CSS Grid `1fr 1fr`）。模板逻辑由 4 个 `.traffic-cell` 组成，结构对称易扩展。
- `showQR(name, region, sub, link)` 函数签名扩展：节点弹窗传 `region`（短地域 badge），备用订阅传 `sub`（客户端示例，渲染为副标题）。
- `.modal-title` 加 `flex-wrap` + `word-break` + `justify-content:center`，极端长名字下也能优雅折行且整体居中。
