# xray-pilot v0.2.4

订阅信息页两处 UI 细节修复。

## 修复

- **累计流量「合计」与「最近更新」改为同一行**，时间靠右对齐，节省纵向空间且视觉重心更平衡。
- **二维码弹窗标题不再被挤换行**：备用订阅弹窗（"通用 Base64 / Clash 系 / Sing-box"）此前把客户端示例 `Hint`（如 `V2RayN / NekoRay / Shadowrocket`）作为 `region-tag` 渲染，导致标题文字被挤压换行。现在 `Hint` 显示为弹窗副标题，主标题独占一行。

## 实现要点

- `showQR(name, region, sub, link)` 函数签名扩展，新增 `sub` 参数承载副标题：
  - **节点弹窗**：传 `region`（短地域 badge）+ 不传 `sub`（自动默认"扫描二维码导入此节点"）
  - **备用订阅弹窗**：不传 `region` + 传 `sub`（客户端示例）
- `.modal-title` 加 `flex-wrap` 与 `word-break`，极端长名字下也能优雅折行而不溢出。

## 截图对照

修复前：备用订阅弹窗的 "V2RAYN / NEKORAY / SHADOWROCKET" badge 把 "通用 Base64 订阅" 标题挤成两列竖排。
修复后：标题独占一行，客户端示例移到副标题位置，视觉与节点弹窗一致。
