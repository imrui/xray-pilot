# v0.1.11

本次版本主要收尾节点状态检测稳定性，并补充节点初始化脚本与文档，方便新节点快速接入。

## 亮点

- 修复节点漂移检测误报：远端配置读取改为更稳的传输方式，并将“读取异常”与“真实漂移”区分记录。
- 错开健康检测与漂移检测的启动时间，缓解 SQLite 在定时任务并发时的锁冲突。
- 新增节点一键初始化脚本 `scripts/node-bootstrap.sh`，支持交互式粘贴管理公钥、安装 Xray、开启 BBR，并自动备份 `sshd_config`。
- README 中补充节点初始化说明，支持和主程序安装一致的 `curl | bash` 用法。
- 用户、节点、协议、分组等管理页面继续优化表格信息层级，并统一补充 `#ID` 展示。

## 适合升级的场景

- 节点同步后偶发被误判为“配置漂移”。
- 定时健康检测与漂移检测同时运行时偶发出现 `SQLITE_BUSY`。
- 需要批量初始化新的云节点，并希望统一执行 SSH、Xray 与 BBR 基础准备。
