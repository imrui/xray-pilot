# xray-pilot v0.4.1

v0.4.0 后续若干体验修复与小优化。

## 新增

- **一键接入按钮换上独立配色**（绿描边 + 绿文字），与「新增节点」实心绿、「同步待处理节点」中性灰之间形成清晰层次
- **分组列表「X 个节点」徽章可点击**，点击直接打开该分组的编辑抽屉

## 修复

- **协议管理抽屉勾选 1 个协议时显示完整编辑界面**：之前左侧只勾选 1 个协议时，右侧只剩孤零零的「填充模板」按钮，没有「一键生成」、没有 settings JSON / 端口输入框；现在勾 1 个等同于激活该协议，勾 2+ 个才进入批量模式
- **一键接入命令不再依赖 sudo 透传环境变量**：之前部分发行版 sudo 默认 `env_reset`，`PANEL_URL` / `INSTALL_TOKEN` 会被丢弃导致脚本启动失败。命令现在不带 sudo，对话框文案明确提示「请以 root 用户执行（普通用户先 `sudo su -` 切换）」

## 升级

无 schema 变更、无破坏性 API 变更，直接覆盖二进制重启。
