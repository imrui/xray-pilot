# xray-pilot v0.1.13

修复 Clash 系客户端从「备用订阅链接」导入时被喂入 V2Ray Base64 内容导致 `invalid yaml` 报错。

## 新增

- **备用订阅链接按客户端分组**：通用 Base64（V2RayN / NekoRay / Shadowrocket）、Clash 系（Mihomo / Clash Verge / ClashX）、Sing-box（sing-box / Hiddify），每条都带客户端示例与复制 / 二维码按钮
- 备用链接统一改用显式 `?format=v2ray|clash|singbox` 参数，复制即贴即用

## 兼容性

- 主订阅链接（`/sub/<token>`）行为不变，仍按 `User-Agent` 智能识别 Clash / Sing-box / V2Ray，并兼容原 `?sub=1` 别名
