# xray-pilot v0.4.2

一键接入闭环：防火墙提醒 + 注册后 SSH 连通性自检。

## 新增

### 面板出网 IP 自动探测

- 后台每小时通过外部探针（默认 `https://api.ipify.org`，可用 `XRAY_PILOT_OUTBOUND_PROBE_URL` 环境变量覆盖）拿到面板出网 IP，写入系统设置
- 系统设置页新增「面板出网 IP」模块，展示自动探测值，并提供手动覆盖入口（多 IP / NAT / 容器场景）
- 一键接入对话框在生成命令后展示「请确保节点防火墙允许面板 IP `X.X.X.X` 通过 SSH 端口访问」的明确提示

### 注册后自动做 SSH 连通性自检

节点装机脚本回填到 panel 后，panel 立刻对节点 SSH 端口做一次 TCP 探针：

- **通**：success 态显示绿色「✅ panel 已可正常 SSH 到节点（延迟 X ms）」，3 秒后自动关闭
- **不通**：success 态显示黄色警告「⚠️ panel 暂时无法 SSH 到节点」+ 具体错误（含面板出网 IP），引导用户放行防火墙后到节点列表手动同步；7 秒后才自动关闭

这填上了 v0.4.0 一键接入的最后一个体验缺口——之前「脚本跑成功」≠「panel 能 SSH 到节点」，用户离开节点终端后才在同步阶段发现防火墙问题；现在装完立刻知道结果。

## 升级

- 数据库 schema 自动迁移（`node_install_tokens` 加 reachable 字段；系统设置新增两个 key），无需手动操作
- 无破坏性 API 变更，可直接覆盖二进制重启

## 注意事项

- 容器 / 反代部署下自动探测拿到的可能是入口 IP，请到系统设置「面板出网 IP > 手动覆盖」填入实际值
- SSH 自检超时 5 秒，节点 SSH 端口非 22 时按节点配置的端口探针
